# E-Commerce App (React + TS + Vite)

Полноценный интернет-магазин с фильтрацией, корзиной и оптимизированным взаимодействием с API.

## 🚀 Ключевые особенности и оптимизации

*   **Оптимизация сетевого трафика (Custom RTK Query Debounce, Zod):** Реализовал кастомную обертку над `baseApi`, debounce-логику на уровне endpoint. Снизил нагрузку на бэкенд при работе с "живыми" фильтрами (Range Slider цены) без лишних ререндеров и сетевых запросов. Реализовал строгую валидацию схем данных (Runtime Validation) через Zod внутри RTK Query. Исключаем появление ошибок из-за некорректных ответов сервера.
*   **Управление состоянием:** 
    *   **Redux Toolkit** для глобального стейта (фильтры, корзина).
    *   **Redux Persist** для сохранения корзины между сессиями пользователя.
*   **Работа с формами:** Валидация и сбор данных заказа через **React Hook Form**.
*   **Типизация:** TypeScript, использование DTO и строгих интерфейсов для бизнес-логики.
*   **Архитектура:** Компонентный подход, декомпозиция бизнес-логики в кастомные хуки и редюсеры.

## 🛠 Стек технологий

- **Core:** React 18, TypeScript, Vite
- **State:** Redux Toolkit, Redux Persist
- **API:** RTK Query (Fetch API), Zod
- **Forms:** React Hook Form
- **Styles:** SCSS Modules, BEM

## 📦 Установка и запуск

Проект использует **json-server** для имитации бэкенда и **concurrently** для одновременного запуска всех сервисов одной командой.

1. Установка зависимостей
```bash
      npm install
```

2. Запуск в режиме разработки
```bash
      npm run dev
```

3. Сборка проекта
```bash
      npm run build
```
